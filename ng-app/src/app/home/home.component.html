Yes, we share five movies and 96 coactors.

Welcome to the page to explore actors’ relationships to each other. Have you ever noticed some actors keep showing up together in the same movie? I was curious about how often that happened and created this app.
Please note this is a DEMO app. If an actor’s or title’s info is not already in database, it may take a few moments to get the data. Those whose data is known have been helpfully tagged with Info known. The others will be imported.

Also, please pardon the response time. First, the frontend and backend servers have to be spun up and I went with the smallest server that looked viable for the database.
